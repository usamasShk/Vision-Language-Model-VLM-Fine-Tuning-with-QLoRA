# Vision-Language-Model-VLM-Fine-Tuning-with-QLoRA
This repository contains a Jupyter Notebook demonstrating how to fine-tune a Vision Language Model (VLM) using QLoRA for the task of converting document images into structured Markdown documentation.

## Objective
- Understand multimodal learning (image + text)
- Perform parameter-efficient fine-tuning (QLoRA)
- Adapt a pretrained VLM to document understanding
- Generate Markdown outputs from unseen document images
